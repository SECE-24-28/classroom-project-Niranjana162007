Chapter 1 – C Programming



1. Print Hello World
#include <stdio.h>
int main() {
    printf("Hello World");
    return 0;
}

2. Sum of Two Numbers
#include <stdio.h>
int main() {
    int a=10, b=20;
    printf("Sum = %d", a+b);
}

3. Even or Odd
int n=7;
if(n%2==0) printf("Even");
else printf("Odd");

4. First 5 Numbers
for(int i=1;i<=5;i++)
    printf("%d ",i);

5. Factorial
int n=5,f=1;
for(int i=1;i<=n;i++) f=f*i;
printf("%d",f);

Chapter 5 – Chemistry


Chemical Equations

  • H₂ + O₂ → H₂O
  • 2Mg + O₂ → 2MgO
  • C + O₂ → CO₂
  • CaCO₃ → CaO + CO₂
  • Fe + S → FeS

Definitions

Atom
The smallest unit of matter.

Molecule
Group of atoms bonded together.

Element
Substance made of one kind of atom.

Compound
Combination of two or more elements.

Chemical Reaction
Process that forms new substances.
